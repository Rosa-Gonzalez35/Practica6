¿Qué es frontend?

Un frontend es la interfaz gráfica de usuario de una página web o una aplicación. El frontend es donde se presenta la información y donde tienen lugar las interacciones del usuario.

Fundamentos de frontend

  • HTML:

    • Nuestro arquitecto digital. HTML (HyperText Markup Language) es como el cimiento de una casa: es la base estructural de todas las páginas web. Sin él, no tendríamos la maravillosa arquitectura de Internet.



  • CSS:

    • El estilista de la web. CSS (Cascading Style Sheets) es quien le da estilo a la estructura de HTML. Es como el diseñador de interiores que embellece una casa. En palabras sencillas, controla todo, desde colores y tipografía hasta la presentación visual.



  • JAVASCRIPT:

    • El genio de la interacción. JavaScript convierte las páginas web estáticas en experiencias interactivas. Si deseas que tus usuarios hagan clic, arrastren, desplieguen y se diviertan, JavaScript es esencial. Los marcos de trabajo (frameworks) como Angular, React y Vue.js son sus mejores amigos, proporcionando herramientas para crear aplicaciones web dinámicas.



  • API:

    • El puente entre mundos. La API (Interfaz de Programación de Aplicaciones) es la clave para conectar diferentes sistemas, software y aplicaciones. Es como un intérprete multilingüe que permite a tus aplicaciones comunicarse entre sí.
